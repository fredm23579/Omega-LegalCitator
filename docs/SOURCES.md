# Sources

This repository is seeded around the following source families requested for the project.

## Included source families
- Web
- Scholar
- GitHub
- Google Drive
- OneDrive
- Social
- Hugging Face
- Google Calendar

## Planned responsibilities
- Web: authority lookup, court/rule pages, open legal corpora
- Scholar: law review and research support for citation context
- GitHub: reusable legal-tech libraries, datasets, and reference implementations
- Google Drive: ingest user-owned legal notes and working documents
- OneDrive: ingest user-owned legal notes and working documents
- Social: monitor public legal discourse and emerging authority references
- Hugging Face: legal datasets, retrieval models, citation tasks, evaluation assets
- Google Calendar: procedural timelines, hearing dates, filing reminders

## Data policy
Every source adapter should emit:
- source type
- source identifier
- retrieval timestamp
- provenance URL or file reference when available
- confidence
